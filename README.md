## notes-app
Sticky notes app with vanilla JS.

### Description
HTML,CSS,Javascript 를 활용하여 간단한 메모앱을 만들어보았습니다. 
HTML요소를 생성하여 DOM에 추가, grid-template, 로컬 스토리지의 전체적인 작동원리에 대하여
효과적으로 학습할 수 있었습니다.

### Features
* "+"를 누를 시 메모장 추가
* 메모장 수정 
* 더블 클릭 시 메모장 삭제
